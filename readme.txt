== Installation ==

1. Unzip the downloaded zip file and upload the plugin folder into the `wp-content/plugins/` directory. Alternatively, upload from Plugins >> Add New >> Upload Plugin.
2. Activate the plugin from Plugins page.

= Usage =

After successfully installing and activating the plugin, you will find "Logo Showcase" menu on left column of WordPress dashboard. Go to Logo Showcase >> Configure any options as desired using "Shortcode Generator" page. Then copy the shortcode which is at the "Shortcodes" tab section and paste it wherever you want to display the carousel slider.
